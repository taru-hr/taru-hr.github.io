How this embed is made (reuse it in any project)

1 · Host the file at a public web address. On GitHub Pages every file in the repo already has one — here it is https://taru-hr.github.io/HR_Operations_Hub%20Taru%20HR.pptx (a space becomes %20).

2 · Wrap that address with the Office viewer and put it in an <iframe>:

https://view.officeapps.live.com/op/embed.aspx?src=YOUR-FILE-ADDRESS

That is the whole technique — it works for .pptx, .docx and .xlsx too. The file must be public, and it uses Microsoft's service.
